 # 📊 Sales Analytics Dashboard (Power BI) 
This project focuses on building an interactive Sales Analytics Dashboard using Power BI. The dashboard helps businesses track key sales metrics such as revenue, profit, product performance and regional sales distribution. Key insights include identifying top-performing products, tracking monthly sales trends and analyze regional sales performance.


An interactive **Sales Analytics Dashboard** built using Power BI to visualize and analyze business performance across multiple dimensions including sales, profit, orders, and product insights.



## 🚀 Dashboard Overview

This dashboard provides a **comprehensive and visually appealing analysis** of sales data with a modern dark theme UI. It helps stakeholders quickly understand key metrics and trends for better decision-making.
